# rust-vscode-starter

A template for starting [Cargo](https://github.com/rust-lang/cargo)-based [Rust](https://www.rust-lang.org/) projects in [VS Code](https://code.visualstudio.com/).


## Extensions

These [VS Code Extensions](https://code.visualstudio.com/docs/editor/extension-gallery) are required for all the features of this repository to work:

1. [rust-lang.rust](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust)
1. [bungcip.better-toml](https://marketplace.visualstudio.com/items?itemName=bungcip.better-toml)
1. [vadimcn.vscode-lldb](https://marketplace.visualstudio.com/items?itemName=vadimcn.vscode-lldb)
1. [editorconfig.editorconfig](https://marketplace.visualstudio.com/items?itemName=editorconfig.editorconfig)

## Features

- Debugging with breakpoints
- Code execution via tasks
- Code formatting via [rustfmt](https://github.com/rust-lang/rustfmt) with custom [configuration](./rustfmt.toml)
- Editor configuration via [.editorconfig](./.editorconfig)
- See the [remaining features](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust) of the Rust extension
